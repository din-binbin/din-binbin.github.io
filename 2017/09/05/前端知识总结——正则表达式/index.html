正则语法

正则表示的是对字符串的一种过滤逻辑，其主要作用是匹配和获取。

正则表达式的创建

类似于创建对象，正则表达式也是一个对象，也可以使用构造函数的方式来创建。

基本语法

pattern 表述 pattern 表述
[xyz] 匹配任意一个包含的字符 [^xyz] 匹配任意一个未包含的字符
\w 匹配字母、数字或者下划线的字符 \W 匹配非字母、数字或者下划线的字符
\s 匹配任意空白符 \S 匹配非空白符的字符
\d 匹配数字 \D 匹配非数字的字符
\b 匹配单词的开始或结束位置 \B 匹配非单词的开始或结束位置
$ 匹配字符串的结束位置 ^ 匹配字符串的开始位置

另外注意

  • . 匹配任意字符，除了换行符
  • 遇到空格，尽量使用 \s

RegExp对象

正则表达式是一个对象，RegExp对象。

其常用方法有：

  • test()
    • 检测字符串是否符合该正则表达式
    • 符合返回true，不符合返回false
  • exec()
    • 返回一个类数组的对象

常用属性有：

  • source
  • ignoreCase
    • flag标志位属性，加上表示忽略大小写
  • global
    • flag标志位属性，加上表示全局匹配
  • multiline
    • flag标志位属性，加上表示多行匹配

量词（重复）

  • ? 重复0次或1次
  • + 重复1次或多次
  • * 重复0次或多次
  • {n} 重复n次
  • {n,} 重复n次或更多次
  • {n,m} 重复n次到m次

在调用exec()这个方法的时候，如果有采用分组的方式定义正则，那么会自动使用捕获，并在返回的类数组对象里返回该捕获。因此，分组和捕获通常联系在一起。

以上所说的默认是捕获型分组，除此之外还有非捕获型分组。

不需要捕获分组的时候，在分组加上 ?: 即可，即为非捕获型分组了。
